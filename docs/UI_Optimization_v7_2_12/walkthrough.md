# 修正内容の確認 (v7.2.12)

## 実施内容
UMIS Pro v7.2.12 において、操作性の向上とUIの最適化を実施しました。

### 1. 起点・終点方式の一括選択機能の刷新
- 範囲選択ロジックをステートマシン方式に刷新しました。
- 範囲選択モードがOFFの時にクリックしたファイルが自動的に「起点」として保存されます。
- モードをONにして別のファイルをクリックすると、その「起点」から現在の「終点」までの全ファイルが一括選択されます。
- ボタンのツールチップを状況に合わせて動的に変更（例：「範囲の終点を選択してください」）するようにし、ユーザーへのガイダンスを強化しました。

### 2. シーングリッドのレスポンシブ化
- スプリットバーのドラッグによるパネル幅の変更に連動して、シーン一覧の列数を動的に変更する機能を実装しました。
- シーンパネルの幅が 250px 未満になった際、自動的に1列表示に切り替えることで、狭い幅でも文字が潰れず視認性を維持できるようになりました。

## 検証結果
- 「起点選択 → モードON → 終点選択」による一括選択が確実に動作することを確認。
- スプリットバー操作時に、リアルタイムでシーングリッドの列数が切り替わることを確認。
