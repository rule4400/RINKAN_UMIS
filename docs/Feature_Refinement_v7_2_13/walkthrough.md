# 修正内容の確認 (v7.2.13)

## 実施内容
UMIS Pro v7.2.13 において、機能の標準化と安全性・利便性の向上を実施しました。

### 1. 一括選択（Shift+Click）の標準化
- 独自の「範囲選択モード」ボタンを廃止し、OS標準に近い Shift + クリック による範囲選択を再実装しました。
- 内部的な起点（`range_selection_start_idx`）管理を最適化し、より安定した選択動作を実現しました。

### 2. 会場選択の永続化
- 会場ドロップダウン (`dd_venue`) で会場を変更した際、即座にプロジェクト設定に保存されるようにしました。
- これにより、アプリの再起動やプロジェクトの切り替えを行っても、最後に選択した会場が自動的に復元されます。

### 3. カード初期化ロジックの厳格化と自動リセット
- 「カード初期化」ボタンの有効化条件を「取り込み完了」または「緊急フォーマットON」に厳格化しました。
- 「緊急フォーマット」を使用して初期化を実行した場合、実行後に自動的に設定を「OFF」に戻し、意図しない連続的な初期化を防ぐ安全策を導入しました。

### 4. UIの微調整
- シーン一覧のレスポンシブ切り替え閾値を 240px に微調整しました。

## 検証結果
- Shiftキーによる範囲選択が正常に動作することを確認。
- 会場選択が永続化され、再起動後も保持されることを確認。
- カード初期化実行後に「緊急フォーマット」設定が自動でOFFになることを確認。
