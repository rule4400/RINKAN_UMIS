# UMIS Pro v7.2.13 機能改善・バグ修正実装計画

## 概要
ユーザー体験の向上と安全性の強化を目的として、一括選択ロジックの再調整、会場設定の永続化、およびカード初期化プロセスの厳格化を行います。

1. **一括選択の標準化**: 独自の「範囲選択モード」を廃止し、標準的な Shift + クリックによる範囲選択を安定動作させます。
2. **会場選択の永続化**: 選択された会場をプロジェクト設定に保存し、再起動後も維持されるようにします。
3. **安全なカード初期化**: 取り込み完了後、または「緊急フォーマット」が明示的にONの場合のみ初期化を許可し、初期化後は「緊急フォーマット」設定を自動でリセットします。

## 変更内容

### UMIS App 本体

## 検証計画
- Shiftキーを押しながらファイルをクリックし、範囲選択が正常に動作することを確認。
- 会場を切り替えた後、アプリを再起動（またはプロジェクト切り替え）して、選択が維持されていることを確認。
- 「カード初期化」ボタンが通常は無効であり、取り込み完了後または「緊急フォーマット」ONで有効になることを確認。
- 「緊急フォーマット」ONで初期化実行後、設定が自動的にOFFに戻ることを確認。
