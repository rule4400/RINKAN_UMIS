# 修正内容の確認 (v7.2.20)

## 実施内容
v7.2.19 で発生していた Shift キーの状態同期漏れを修正し、標準的な Shift+クリックによる範囲選択機能を完全に復旧させました。

### 1. Shiftキー状態同期の修正
- `on_keyboard` イベントの冒頭で `self.shift_pressed = e.shift` を実行するように修正しました。
- これにより、Shiftキーが押されている間は確実に `self.shift_pressed` が True となり、デバッグ表示も [SHIFT ON] と連動するようになりました。

### 2. 左クリック範囲選択ロジックの最適化
- `on_file_click` 内の範囲選択ロジックを刷新し、Shiftキーが押されている場合の一括選択処理を確実に実行するようにしました。
- 実行後には「Shift Mode Reset」と表示し、フラグをリセットすることで、予期せぬ連続選択を防ぐ安全策を講じました。

### 3. 互換性の維持
- 前バージョンで導入した「右クリック（副ボタン）による範囲選択」も引き続き有効であり、Shiftキーが効きにくい環境でも代替手段として利用可能です。

## 検証結果
- Shiftキー単体を押した際、ツールバーのラベルが [SHIFT ON] に切り替わることを確認。
- Shiftキーを押しながらサムネイルを左クリックすることで、範囲内が一括選択されることを確認。
- 右クリックによる範囲選択も意図通り動作することを確認。
