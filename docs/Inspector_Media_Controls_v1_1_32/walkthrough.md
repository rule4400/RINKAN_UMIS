# 実装完了報告 (Walkthrough)

セレクトモードのインスペクタにおいて、動画および音声ファイルの再生をコントロールするためのUIとロジックを実装しました。

## 変更内容

- **UIの追加**:
  - インスペクタのプレビューエリア直下に、再生/一時停止ボタン、シークバー（スライダー）、再生時間表示を追加しました。
  - 画像ファイルなどの非メディアファイル選択時には、これらのコントロールは自動的に非表示になります。

- **再生位置の同期**:
  - `ft.Video` および `ft.Audio` の `on_position_changed` イベントを利用し、再生に合わせてシークバーがリアルタイムに更新されるようにしました。
  - 再生時間（現在位置 / 全体長さ）を `00:00 / 00:00` 形式で表示します。

- **シーク機能**:
  - シークバーをドラッグして放すと、その位置に再生がジャンプします。
  - ドラッグ中は自動更新によるスライダーの戻りを抑制しています。

- **再生状態の同期**:
  - 再生/一時停止ボタンのアイコンが、実際のメディアの再生状態に合わせて動的に変化します。
  - 音声の場合は `on_state_changed`、動画の場合は `on_play` / `on_pause` イベントで同期しています。

## 検証結果

- **動画**: シークバーの同期、シーク操作、ボタンによる再生/停止が正常に動作することを確認しました。
- **音声**: 音声再生中もシークバーと時間が更新され、操作が可能であることを確認しました。
- **画像**: インスペクタにメディアコントロールが表示されず、従来のメタデータ表示のみになることを確認しました。
