# セレクトモード・インスペクタへのメディアコントロール実装計画

セレクトモードのインスペクタで動画や音声を再生する際、再生位置の確認と操作（シーク）ができるように、シークバー（スライダー）と再生/一時停止ボタンを追加します。

## ユーザーレビューが必要な事項

- **操作感**: 標準の `ft.Video` コントロール（`show_controls=True`）を表示したままにするか、独自コントロールのみにするか。今回は独自コントロールを追加し、操作の一貫性を持たせます。

## 提案される変更

### [RinkanUMISApp クラス]

#### [MODIFY] `__init__`
- メディア操作用のUIコンポーネント（スライダー、再生ボタン、時間表示）を初期化します。
- `self.inspector_media_controls` というコンテナにまとめます。

#### [MODIFY] `build_ui`
- インスペクタの構築時に `self.inspector_media_controls` を配置します。

#### [MODIFY] `_update_inspector_details`
- 選択されたファイルが動画または音声の場合に `self.inspector_media_controls` を表示し、それ以外（画像等）では非表示にします。

#### [MODIFY] `_build_preview_widget`
- `ft.Video` および `ft.Audio` にイベントハンドラ（`on_position_changed`, `on_duration_changed`, `on_play`, `on_pause` 等）をアタッチします。

#### [NEW] イベントハンドラ
- `_on_media_pos_change`: 再生位置が変更されたときにスライダーと時間を更新します。
- `_on_media_dur_change`: メディアの長さが確定したときにスライダーの最大値を設定します。
- `_on_media_seek_start` / `_on_media_seek_end`: ユーザーによるスライダー操作（シーク）を処理します。
- `_toggle_inspector_media_play`: インスペクタ上の再生/一時停止を切り替えます。

## 検証計画

### 手動確認
1. セレクトモードに切り替える。
2. 動画ファイルを選択する。
   - インスペクタにプレビューとシークバーが表示されることを確認。
   - 再生中にシークバーが動くことを確認。
   - シークバーをドラッグして再生位置が飛ぶことを確認。
   - 再生/一時停止ボタンが正しく機能することを確認。
3. 音声ファイルを選択する。
   - 同様にシークバーとボタンが表示され、操作できることを確認。
4. 画像ファイルを選択する。
   - シークバーが非表示になることを確認。
