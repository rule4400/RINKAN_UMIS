# セレクトモード：シーン移動機能の実装 (v10.7.0)

セレクトモードにおいて、誤って別のシーンに取り込まれたファイルを正しいシーンへ移動・リネームする機能を実装しました。

## 実施済みタスク
- [x] インスペクタへの「別のシーンへ移動」ボタンの追加
- [x] 移動先シーンを選択するダイアログの実装
- [x] セレクトモードへの「一括選択モード」の追加
- [x] セレクトモード読み込み時の全選択を廃止（デフォルト未選択に変更）
- [x] チェックマークを四角形（CHECK_BOX）に変更
- [x] アプリ最下部に常駐するステータスバーを実装
- [x] サムネイル生成、スキャン、シーン移動などの各タスクと状態表示を連携
- [x] ステータスバー右端に製作者クレジット（Developed by Keizo Ando）を表示
- [x] インスペクタのスプリッターに「ハンドル（中心線）」を追加し、視認性を向上
- [x] スプリッターの当たり判定（幅）を拡大し、操作しやすく改善
- [x] セレクトモードのグリッドを中央寄せ（CENTER）にし、インスペクタ調整時の不自然な余白を解消
- [x] インスペクタの幅変更に連動してグリッドが即座にリフロー（再配置）されるように改善
- [x] セレクトモードのアイテムを左詰め（START）に戻しつつ、グリッド全体を中央寄せに配置
- [x] これにより、最終行のアイテムが不自然に中央に寄る問題を解決し、かつ全体のバランスも維持
- [x] セレクトモードにおいて、チェックボックスで選択したアイテムに青枠が残る問題を修正
- [x] 青枠（フォーカス）は、現在インスペクタで表示している1つのアイテムのみに限定されるよう厳格化
- [x] インスペクタ（右サイドバー）の最大幅をウィンドウ幅の50%まで拡大できるように改善
- [x] これにより、大画面モニター等でプレビューをより大きく表示することが可能に
- [x] セレクトモードにて矢印キーで順送りする際、非表示（フィルタアウト）にしているカテゴリのファイルをスキップするように改善
- [x] これにより、フィルタリング中も意図した通りの順番でプレビュー確認が可能に
- [x] 新しいファイル（v10.7.11.py）として保存・管理
- [x] 選択状態の視覚効果の強化（チェックアイコンとボーダー）
- [x] 物理移動およびファイル名のリネーム（シーン名置換）ロジックの実装
- [x] 移動先でのカテゴリフォルダ（動画/写真等）の自動生成と振り分け
- [x] 選別済みフォルダ（選別/）内のファイルの同期移動対応
- [x] 不具合修正（NoneTypeエラー、変数の未定義エラー）
