# 修正内容の確認 (Walkthrough) - v10.7.0/10.7.2

## 概要
セレクトモードにおいて、シーンの割り当てミスを修正するための「シーン移動」機能を実装しました。物理的なファイルの移動に加え、ファイル名の置換、フォルダの自動生成、選別済みフラグの同期など、現場でのミスを即座に修正できる強力なツールとなっています。

## 変更点
### 1. 「一括選択モード」の導入
- セレクトモードのヘッダーに**チェックボックス型の切替ボタン**を追加しました。
- **通常モード**: クリックでファイルをフォーカスし、プレビューを表示します。
- **一括選択モード**: Googleフォトのように、クリックでファイルを選択（チェック）できます。
- **範囲選択**: インジェストモードと同様、**起点をクリック、終点を右クリック**することで、その間のファイルを一括で選択できます。

### 8. キーボードナビゲーションのフィルタ対応 (v10.7.11)
- **非表示アイテムのスキップ**: セレクトモードでカテゴリフィルタ（動画のみ表示など）を使用している際、矢印キーで前後移動しても、現在非表示になっているカテゴリのファイルがプレビューされないように改善しました。
- **直感的な操作感**: 画面に表示されている順番通りにキーボードで順送りできるため、フィルタリングしながらの選別作業がよりスムーズになりました。

### 2. シーン移動機能の実装
- インスペクタ（右サイドバー）に「**別のシーンへ移動**」ボタンを追加しました。
- 移動先のシーンを選択すると、以下の処理がバックグラウンドで実行されます。
  - **物理移動**: 指定したシーンの正しいカテゴリフォルダ（動画/写真/Raw等）へ移動。
  - **自動リネーム**: ファイル名に含まれる旧シーン名（例: `102_朝食`）を新シーン名に自動置換。
  - **フォルダ自動生成**: 移動先にカテゴリフォルダやカメラマンフォルダがない場合、設定に基づき自動作成。
  - **選別同期**: `選別/` フォルダ内のファイルも連動して移動・リネーム。

### 3. 不具合修正と堅牢化
- シーン情報が一部欠落しているファイルでもパスから推測して処理を継続するよう修正（`TypeError` の解消）。
- エラーハンドリング時の変数定義エラー（`UnboundLocalError`）を解消。
