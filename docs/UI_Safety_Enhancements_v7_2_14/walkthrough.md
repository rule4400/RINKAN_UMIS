# 修正内容の確認 (v7.2.14)

## 実施内容
UMIS Pro v7.2.14 において、UIの安定性向上、安全機能の強化、および履歴画面の刷新を実施しました。

### 1. シーングリッドの列数制御の適正化
- `GridView` から `max_extent` を削除し、`runs_count` による直接制御に変更しました。
- スプリットバー操作によるパネル幅の変更時、250pxを境に確実に 1列/2列 が切り替わるようになり、表示の崩れを解消しました。

### 2. リスト表示の視認性向上
- メタデータ列（カテゴリ、サイズ、シーン名）の幅を、フォントサイズ連動から「固定幅」に変更しました。
- ファイル名のみを可変（`expand=True`）とすることで、表示サイズを大きくしても重要なメタデータが画面外に押し出されず、常に見える状態を維持しました。

### 3. 初期化時の未取り込み警告（安全装置）
- カード初期化ボタン押下時に、シーン未設定（＝取り込み対象外）のファイルが存在するかチェックするロジックを追加しました。
- 該当ファイルがある場合、「取り込みがされていないファイルがあるようですが、初期化を実行しますか？」という警告ダイアログを表示し、データの誤消去を二重に防止します。

### 4. 履歴画面 UI の大幅刷新
- 各履歴エントリを `ft.Card` 形式に変更し、情報のまとまりを強化しました。
- 「初期化済 / 未初期化」を判別するバッジをヘッダーに追加し、カードのステータスが一目で分かるようにしました。
- エラーログの表示スタイルも改善し、背景色付きのコンテナで詳細を確認しやすくしました。

## 検証結果
- スプリットバーによるシーングリッドの 1列/2列 切り替えが正常に動作することを確認。
- リスト表示のメタデータが見切れないことを確認。
- 未割当ファイルがある状態で初期化を試み、警告ダイアログが表示されることを確認。
- 履歴画面で新しいカード形式と「初期化済」バッジが正しく表示されることを確認。
