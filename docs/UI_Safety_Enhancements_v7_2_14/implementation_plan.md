# UMIS Pro v7.2.14 UI修正および機能強化実装計画

## 概要
UIの安定性向上、誤操作防止のための安全装置の追加、および履歴画面の視認性改善を目的としたアップデートを行います。

1. **シーングリッドの修正**: `max_extent` を廃止し `runs_count` による確実な列数制御を実装します。
2. **リスト表示の固定幅化**: メタデータ列に十分な固定幅を持たせ、文字見切れを解消します。
3. **初期化時の未取り込み警告**: 未取り込みファイルがある場合に警告ダイアログを表示し、データの誤消去を防ぎます。
4. **履歴UIの刷新**: 「初期化済み」バッジの追加を含め、履歴情報をカード形式で整理して表示します。
