# RINKAN UMIS v1.1.0 修正内容の確認

## 変更概要
インジェストモードとセレクトモードのUI構造を完全に統一し、シームレスな切り替えを実現しました。

### 実施した主な変更
1.  **3カラム構造の統一**:
    - 全モードで「サイドバー（左）＋ メインコンテンツ（中央）＋ インスペクタ/パネル（右）」のレイアウトを採用。
2.  **インジェスト設定のサイドバー移動**:
    - ヘッダーに散らばっていたプロジェクト、会場、ソースドライブの選択UIを左サイドバーに集約。
3.  **インスペクタ風デザインの適用**:
    - インジェストモードのシーン割当パネルを、セレクトモードのインスペクタと同じタイトルバー付きカード形式に変更。
4.  **グローバルヘッダーの簡素化**:
    - モード固有のコントロールを排除し、ロゴとモード切替のみのクリーンなヘッダーに。

### テスト結果
- アプリケーションが正常に起動し、バージョンが 1.1.0 であることを確認。
- インジェスト/セレクトを切り替えても、左側のサイドバーと右側のパネルの幅が維持され、レイアウトのガタつきがないことを確認。
- サイドバー内のドロップダウンからプロジェクトや会場が正しく変更できることを確認。
- 右側の「シーン割当」パネルからシーンの選択や追加が正常に行えることを確認。
