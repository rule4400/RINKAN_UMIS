# 動画の再生時間バッジが表示されない問題の修正 (v1.1.31)

一部のファイルで動画の再生時間バッジが生成されない原因を調査した結果、サムネイルをUIに追加する処理（`_append_thumbnails_to_ui` メソッド）において、バッジのコンテナ(`ft.Container`)ではなく、内部のテキスト要素(`ft.Text`)をUI管理用のリスト(`_thumb_controls`)に渡してしまっていることが原因でした。

これにより、非同期で動画の再生時間が取得された際に、テキストの値は更新されるものの、親であるコンテナの表示状態(`visible`)が更新されず、結果としてバッジが隠れたままになっていました。

## Proposed Changes

新しいバージョン（v1.1.31）としてファイルをコピーし、以下の修正を行います。

### [NEW] `rinkan_umis_v1.1.31.py`
`rinkan_umis_v1.1.30.py` をコピーして作成します。

#### [MODIFY] `rinkan_umis_v1.1.31.py`
- バージョン番号の更新: `VERSION = "1.1.30"` -> `VERSION = "1.1.31"`
- `_append_thumbnails_to_ui` メソッド内の修正:
  - リスト表示用のバッジコンテナを変数 `dur_badge_list` として定義。
  - `_thumb_controls[i]` に格納する要素を `dur_text_grid`, `dur_text_list` から `dur_badge_grid`, `dur_badge_list` に変更。

## Verification Plan
1. 修正後のファイルでアプリを起動し、セレクトモードで動画ファイルが複数含まれるフォルダを読み込む。
2. リスト表示およびグリッド表示の両方で、スクロールして後から読み込まれた動画にも正しく再生時間バッジが表示されることを確認する。
