# UMIS Pro v7.2.10 UI改修・バグ修正実装計画

## 概要
v7.2.9 で発生した起動時のクラッシュバグの修正と、リスト表示におけるレイアウトの微調整を行います。

1. **スプリッターの修正**: Flet 0.27.6 でサポートされていない `hover_color` を削除し、`ft.Container` を使用した互換性の高い実装に変更します。
2. **リスト表示の固定幅化**: メタデータ（カテゴリ、サイズ、シーン名）の幅を固定値に戻し、スライダーによる拡大時もレイアウトが崩れないようにします。

## 変更内容

### UMIS App 本体

#### [MODIFY] [rinkan_umis_pro_v7.2.10.py](file:///Users/kiroku_keizo/開発/UMIS%20(Universal%20Media%20Ingest%20System)/安定版/rinkan_umis_pro_v7.2.10.py) [NEW]

- **VERSION更新**: `VERSION = "7.2.10"` に更新。
- **UI構築 (`build_ui`)**:
    - `splitter` の定義を `ft.VerticalDivider` から `ft.Container` ベースに変更。
    - `hover_color` 引数を削除し、`on_hover` イベントで背景色を変更するロジックを追加。
- **リスト表示更新 (`_refresh_list_view`)**:
    - メタデータ列の幅を固定値に変更。
        - カテゴリ: `width=55`
        - サイズ: `width=70`
        - シーン名: `width=100`

## 検証計画
- アプリケーションがクラッシュせずに起動することを確認。
- スプリッターにマウスを乗せた際の色変更（ホバー効果）を確認。
- リスト表示でスライダーを動かした際、ファイル名以外の情報が画面外に押し出されないことを確認。
